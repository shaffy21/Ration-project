Security Architecture

Password Security
SHA1
SHA-1 Hashing with per-user random salts in pwd_salt
🔐
Salted Hash prevents rainbow table attacks
🔒
Account Lockout after repeated failed attempts (failed_attempts column)
Data Protection
🪪
Aadhaar Masking — last 4 digits shown in UI; full number stored encrypted
🎫
Time-limited Tokens — auto-expire after 2 hours, Admin-revocable
📜
RBAC — Admin / Dealer / Citizen role checks enforced in every stored procedure
